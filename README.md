## 📌 О проекте

Коллекция полезных модулей для **Hikka UserBot** с готовыми решениями для автоматизации и кастомизации Telegram профиля. Все модули легко устанавливаются и настраиваются.

---

## 📦 Модули

- **🕐 FancyClock** — отображение времени в нике с красивыми стилями
- *Больше модулей в разработке...*

## 🚀 Быстрый старт

```
.install boneseo/bones-
```

Или добавьте репозиторий вручную в конфиг Hikka.
