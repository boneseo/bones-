## 📋 Модули

### 🕐 FancyClock
Модуль для отображения текущего времени в имени профиля с красивыми стилями цифр.

**Особенности:**
- ⏳ Автоматическое обновление времени в нике (каждую минуту)
- 🎨 10+ стилей оформления цифр:
  - **Default** — обычные цифры
  - **Bold Math** — полужирные математические
  - **Bold Sans** — полужирные без засечек
  - **Bold Italic** — полужирные курсивные
  - **Mono** — моноширинный шрифт
  - **Fullwidth** — полноширинные
  - **Superscript** — надстрочные
  - **Subscript** — подстрочные
  - **Circle** — в кружках
  - **Black Circle** — чёрные кружки
- 🌍 Выбор часового пояса по городу/стране
- ⏰ Отключение часов в указанное время
- 🔄 Сохранение настроек

## 📖 Команды

| Команда | Описание | Пример |
|---------|---------|--------|
| `.fc` | Включить/выключить часы в нике | `.fc` |
| `.fcf` | Выбрать стиль шрифта | `.fcf` |
| `.fcs` | Установить часовой пояс | `.fcs Москва` |
| `.fct` | Выключить часы в указанное время | `.fct 22:30` |

### Примеры использования

**Включить часы:**
```
.fc
```
> ⏳ Часы включены

**Выбрать стиль шрифта:**
```
.fcf
```
> Откроется меню с выбором 10+ стилей

**Установить часовой пояс по городу:**
```
.fcs Красноярск
```
> ✅ Красноярск, Россия
> Таймзона: Asia/Krasnoyarsk
> Время: 𝟙𝟡:𝟛𝟝

**Выключить часы в определённое время:**
```
.fct 22:30
```
> ⏰ Часы выключатся в **22:30**

## ⚙️ Особенности

### Работа с профилем
- Часы обновляются каждую минуту
- При отключении часов восстанавливается оригинальное имя
- Автоматический выбор таймзоны при установке города
- Защита от flood-ограничений Telegram

### Поддерживаемые таймзоны
Модуль поддерживает все стандартные часовые пояса (IANA). Просто укажите город или страну:

```
.fcs Москва      → Europe/Moscow
.fcs Нью-Йорк    → America/New_York
.fcs Токио       → Asia/Tokyo
.fcs Австралия   → зависит от города
```

## 💻 Разработка

### Структура кода
```
FancyClock.py
├── FancyClockMod (основной класс модуля)
├── Команды (.fc, .fcf, .fcs, .fct)
├── Служебные функции (_fancy, _loop, _set_name)
└── Константы (FONT_LIST, DEFAULT_TZ, эмодзи)
```

### Расширение функциональности

Если вы хотите добавить свой шрифт, отредактируйте массив `FONT_LIST`:

```python
FONT_LIST = [
    ("Default",     "0123456789"),
    ("Ваш шрифт",   "нужны 10 символов"),
    # ...
]
```
